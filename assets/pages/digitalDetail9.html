[TrackA]Power BI활용 시각화 과정

교육과정 특징

  • · Power BI가 무엇인지 이해합니다.
  • · Power BI를 활용한 데이터 분석, 시각화 방법 습득합니다.
  • · Power BI로 분석한 컨텐츠를 관리하고 협업을 진행하는 방법을 습득합니다.

교육과정 로드맵

  • 1단계
    Power BI Overview

  • · BI 및 Power BI의 이해

    · Power BI요소 및 라이선스, 아키텍쳐 학습

    · Power BI서비스에서 데이터 분석방법 학습

  • 2단계
    Power BI Desktop

  • · Power BI Desktop의 이해 및 설치

    · 데이터 가져오기 및 전처리 작업 수행

    · 데이터 모델링 및 DAX를 활용한 분석식 작성방법 학습

  • 3단계
    시각화 보고서 작성

  • · 시각화 작업 순서 인지

    · 보고서 페이지 설정 및 언바운드 개체 작성방법 학습

    · 차트작성, 데이터 정렬, 맵, 테이블 등 기능 학습

  • 4단계
    Power BI Service & Power BI Mobile

  • · 작업 영역의 이해 및 웹 게시물 제작

    · 대시보드 작성 및 컨텐츠 공유 방법 학습

    · Power BI Mobile 활용법 습득

  • 5단계
    고급 시각화 기법

  • · 조건부 서식을 활용한 고급 서식 지정 방법 학습

    · 시각적 상호작용, 드릴스루, 드릴다운 설정 방법 학습

    · 사용자 지정 시각적 개체 활용법 인지

  • 6단계
    Power Query 고급 활용

  • · 다양한 유형의 데이터 원본 가져오기 기능 학습

    · 다양한 데이터 취합 방법 및 데이터 전처리 학습

    · 데이터 새로 고침 기능 실습

  • 7단계
    데이터 모델링

  • · 데이터 모델링 관계의 이해

    · 카디널리티 및 필터 방향의 이해

    · DAX를 활용한 데이터 분석 학습

  • 8단계
    데이터 분석 프로젝트

  • · 개인별 데이터 분석 Project 과제 선정

    · 보고서 작성을 위한 스토리 보드 제작

    · 분석 보고서 작성
