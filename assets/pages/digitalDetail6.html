[공통 Track] 데이터 분석 및 시각화 과정 엑셀(심화)

교육과정 특징

  • · Excel BI를 활용한 효과적인 데이터 관리 및 분석이 가능하게 됩니다.
  • · 데이터 분석 프로젝트를 위한 ‘데이터 분석 프로젝트 기획서’ 제공하며 데이터 분석 프로젝트 결과물을 평가할 기준 및 평가방법을 학습합니다.

교육과정 로드맵

  • 1단계
    데이터 전처리를 위한 기능

  • · 효율적인 데이터 전처리를 위한 핵심 기능 학습

    - 피벗 테이블로 데이터 추출하기

    - 텍스트 나누기로 데이터 분리하기

    - 중복된 항목 제거하기

    - 필터와 고급 필터의 활용

    - 데이터 유효성 검사의 이해와 활용

  • 2단계
    데이터 시각화

  • · 효과적인 데이터 시각화를 위한 차트 작성 및 조건부 서식 학습

    - 기본 차트 작성하기/ 심화 차트 작성하기

    - 조건부 서식의 이해와 활용

  • 3단계
    통계적 데이터 분석

  • · 엑셀에서 가능한 수준의 통계적 데이터 분석 기법 학습

    - 통계학의 개념과 목적

    - 기초 통계 상식/ 상관분석과 회귀분석

    - t-test/ 시계열 데이터 분석

  • 4단계
    데이터 분석 프로젝트

  • · 실제 데이터 분석 수행을 통해 각 과정에 맞는 엑셀 활용법 체득과 인사이트 도출

    - 프로젝트(1) LA Lakers 3점 슛 데이터 분석

    - 프로젝트(2) 손익계산서 작성 및 손익 데이터 분석터

    - 프로젝트(3) Super store 데이터 분석과 대시보드 작성

  • 5단계
    데이터 분석 종합 평가

  • · 엑셀을 활용한 데이터 분석의 종합적 이해도, 실제 사용 여부 평가
