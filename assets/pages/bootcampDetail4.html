4. Spring MVC 및 Spring Boot

정의

  • · spring Framework 대한 이해 제시
  • · 웹서비스 개발과 web Framework의 이해

편성 배경

  • · java와 Spring Boot는 현업에서 가장 인기있는 프로그래밍 언어 및 웹 프레임워크이자, 가장 많은 채용 수요를 보유
  • · 개발 생산성을 높여주는 다양한 기능과 라이브러리를 제공하며, 개발자가 더 적은 코드로 빠르게 웹 애플리케이션을 구축하는 데에 도움.
  • · 최근 대규모 기업들이 java 및 Spring Boot를 주력 기술로 채택 중이며, 마이크로서비스 아키텍처 및 클라우드 네이티브 애플리케이션 개발에도 활용됨.

학습목표

  • 목표
    표준 프레임워크 지원

  • · 다양한 오픈소스 프로젝트가 활발하게 개발되고 있으며, java와 Spring boot은 많은 기업에서 표준으로 사용됨.

    · Spring Boot는 Spring framework의 기능을 더욱 강화 및 간소화 한 것으로, 스프링 기반의 애플리케이션의 더욱 빠른 개발을 지원함

    · java와 Spring boot 학습을 통해, 산업에서 널리 사용되는 기술 습득 및 높은 생산성과 안정성을 지닌 애플리케이션을 개발할 수 있음.
