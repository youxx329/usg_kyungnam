[공통 Track] 데이터 분석 및 시각화 과정 엑셀(기본)

교육과정 특징

  • · 다량의 데이터를 효과적으로 분석할 수 있는 엑셀의 다양한 기능을 이해하고 활용하여 데이터 기반 의사 결정력을 강화할 수 있습니다.
  • · 엑셀의 기능을 잘 익힐 수 있는 예제를 활용하여 데이터 분석을 위한 엑셀의 기능을 활용하는 방법을 익힙니다.

교육과정 로드맵

  • 1단계
    데이터 분석의 정의

  • · 데이터 분석의 프로세스 및 주요 사례의 확인

    - 데이터 분석의 개념과 과정과 데이터 분석의 중요성과 사례

    - 우리가 엑셀을 배워야 하는 이유

  • 2단계
    엑셀과 데이터 분석 기초

  • · 엑셀의 기본 원리 학습

    - 엑셀의 기본 원리

    - 빠른 실행도구 모음과 단축키

  • 3단계
    탐색적 데이터 분석

  • · 탐색적 데이터 분석의 개념과 엑셀을 활용한 EDA 방법 이해

    - 탐색적 데이터 분석의 개념과 중요성

    - 데이터의 형태 파악하기

    - 데이터 분석 도구 활용하기

    - 산점도 등 그래프 작성하기

  • 4단계
    데이터 전처리를 위한 함수 및 기능

  • · 효율적인 데이터 전처리를 위한 핵심 함수 학습

    - IF함수를 활용해 데이터 분류하기

    - COUNT 함수 계열로 데이터 개수 파악하기

    - VLOOKUP 함수의 기초와 응용

    · 효율적인 데이터 전처리를 위한 핵심 기능 학습

    - 텍스트 나누기

    - 텍스트 나누기

    - 중복항목 제거하기

    - 필터 활용 및 데이터 유효성 검사

  • 5단계
    데이터 분석/모델링 및 시각화

  • · t-test를 활용한 가설 검정

    · 회귀분석의 개념과 활용/ 시계열 데이터 분석

    · 효과적인 차트 작성법과 활용/ 조건부 서식의 이해와 활용
