11.MFC프로그래밍

정의

  • · MFC(Microsoft Foundation Class)는 Microsoft에서 제공하는 소프트웨어 개발 프레임워크로, 주로 Windows 기반 응용 프로그램을 개발하는 데 사용
  • · MFC는 C++ 기반의 라이브러리이며, Windows API를 추상화하여 개발자들이 더 쉽게 GUI(Graphical User Interface) 기반 애플리케이션을 개발할 수 있도록 지원

편성 배경

  • · MFC는 윈도우를 제어하는 능력이 높은 더대 함수들의 집합으로서 특히 제조업 분 야에서 활용성이 높음
  • · 기본적으로 자동화설비 구동 SW제어 등 분야에서 C++,C#과 더불어 수요가 많음. 특히 지역 제조업 산업 수요가 높아 취업시 필요
  • · C++는 프로그래밍의 기본 개념부터 시작하여 고급 주제까지 다룰 수 있는 깊이와 넓이를 가지고 있고, 강력한 온라인 커뮤니티와 풍부한 학습 자원을 갖추고 있어 학습자들이 쉽게 도움을 받을 수 있음
  • · C++는 지속적으로 발전하고 있어, 최신 기술 트렌드를 반영한 학습이 가능

학습목표

  • 목표
    객체지향 프로그래밍을 위한 C++의 기본 지식과 기술

  • · C++의 기본 구조와 문법을 이해하고, 효과적인 코드 작성 방법을 습득

    · C++의 객체 지향적 특징을 이해하고, 객체 지향 프로그래밍의 핵심 개념을 실제 코드에 적용하는 방법 습득

    · C++의 고급 기능을 이해하고, 복잡한 프로그래밍 문제를 해결하는 데 필요한 고급 기술을 습득
