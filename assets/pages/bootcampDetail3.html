3. JAVA

정의

  • · java 프로그램 실행구조에 대한 개념 이해
  • · java 변수 사용에 대한 제시

편성 배경

  • · java는 현업에서 지속적으로 널리 사용되는 프로그래밍 언어
  • · 현대 플랫폼 비즈니스 및 산업 디지털 트윈 분야에서 활용도 증대
  • · 입문자를 위한 체계적인 java교육이 IT 분야 기초 지식 습득에 핵심.
  • · java의 객체지향 프로그래밍과 예외 처리 기법이 SW품질 향상에 기여
  • · 실무적 문제 해결 능력 향상을 위한 java실습과 프로젝트 중심의 교육 방향 을 설정하는 것이 필요함.

학습목표

  • 목표
    java 프로그래밍 기초와 응용

  • · java 기본구조와 문법의 철저한 이해: java프로그래밍의 실행 구조 및 변수 사용법을 포함한 기본 문법에 대한 깊이 있는 학습 제공

    · 객체지향프로그래밍(OOP)를 java에서 실제 적용: 객체지향 개념을 java로 실현하는 클래스 제작, 객체 생성, 메서드 활용에 대한 실습 중심 교육

    · java의 고급 특성 이해와 실습: java에서의 상속, 내부 및 익명 클래스 활용과 예외처리, 입출력 처리방법에 대한 심화 학습과 실제코드 작성 실습
